8. 제어문

Posted on 2020-08-25 In fastcampus , javascript 
🐞 1) 조건문 2) 반복문 3) 기타

종류

  1. 조건문 : if else문, switch문
  2. 반복문: for문, while문, do-while문
  3. 기타: break문, continue문

1) 조건문

1.1. if…else문

  • 조건식은 불리언타입(T/F)으로 평가되어야 함

  • else문과 else if문은 옵션

  • 대부분의 if…else문은 삼항연산자로 바꿔 쓸 수 있음

      💡 삼항연산자
  조건식 ? 참이면 이 코드 실행 : 거짓이면 이 코드 실행

1.2 switch문

  • 조건식은 주로 변수가 많음
  • 조건이 많은 경우 if문보다 더 유리
  • 주의: break문이 선언되지 않으면 멈추지 않고 계속 수행됨(fall through)
  • 해당하는 조건이 없는 경우를 대비해 default문을 넣어줌 (선택사항)
    default문은 break필요없음

2) 반복문

  • 반복문에서는 무한반복으로 인해 교착상태에 빠지지 않도록 조건을 잘 점검할 것!

2.1 for문

  • 예제

2.2 while문

  • 조건이 참(true)일때까지 블록 수행

2.3 do-while문

  • 조건이 거짓(false)일때까지 블록 수행
  • 적어도 한 번 수행하고 나서 블록의 마지막 조건을 체크
    ex) 네이버 검색시 카페회원 아니어도 검색글 한 번은 읽기 가능

3. 기타

3.1. break문

  • 선언된 곳의 블록에서 탈출 - 레이블문, 반복문, switch문

3.2 continue문

  • 반복문의 코드 블록 실행을 현 지점에서 중단하고 반복문의 증감식으로 실행 흐름을 이동
