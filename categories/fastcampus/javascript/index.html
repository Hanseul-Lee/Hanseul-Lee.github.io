19. 프로토 타입 (2)

Sep 26, 2020 in Fastcampus, javascript

JavaScript는 흔히 프로토타입 기반 언어이며 객체를 상속하기 위하여 프로토타입이라는 방식을 사용한다. 이는 모든 객체들이 메소드와 속성들을 상속 받기 위한 템플릿으로써 프로토타입 객체(prototype object)를 가진다는 의미이다. 프로토타입 객체도 또 다시 상위 프로토타입 객체로부터 메소드와 속성을 상속 받을 수도 있고 그 상위 프로토타입 객체도 마찬가지입니다. 이를 프로토타입 체인이라 부르며 다른 객체에 정의된 메소드와 속성을 한 객체에서 사용할 수 있도록 하는 근간이 된다.
